# 🦁 Kids Learn & Play App

**Đề tài:** Ứng dụng giải trí tương tác và giáo dục cho trẻ em trên Android.
**Sinh viên:** Nguyễn Đình Khánh - 65131460

## 🚀 Tính năng chính
- **Khám phá:** Thẻ học tập trực quan bằng âm thanh và hình ảnh (Động vật, Giao thông, Chữ cái).
- **Đố vui:** Mini-game nghe tiếng đoán hình, kiếm "Ngôi sao" đổi thưởng.
- **Truyện Audio:** Kho truyện cổ tích online. Tích hợp tính năng dùng Ngôi sao để mở khóa truyện mới.
- **Yêu thích:** Lưu trữ truyện Offline để nghe không cần mạng.
